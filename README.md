# AI Assistant Usage – Excel Dashboard Project

##  Project Overview
This project analyzes how students use AI assistants across different disciplines and study levels.  
The dashboard focuses on understanding **usage patterns, engagement, and outcomes** through meaningful KPIs and clear visual storytelling.

The entire project is built using **Microsoft Excel**, highlighting practical data analysis skills without relying on external tools.

---

##  Key Insights
- **10,000 total sessions** analyzed
- **76.2% success rate**
- **Average session duration:** 20 minutes
- **70% reused sessions**, indicating strong reliance on AI tools
- Successful sessions used more prompts, showing deeper engagement
- Majority of sessions fall under short-duration usage
- Consistent usage trend with peak engagement months
